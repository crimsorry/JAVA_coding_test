## 내 풀이

* HashMap 와 class 를 이용한 풀이

```python
import java.util.*;

class Solution {
    
    private static class User{
        String command;
        String uuid;
        
        User(String command, String uuid){
            this.command = command;
            this.uuid = uuid;
        }
    }
    
    public String[] solution(String[] record) {
        ArrayList<User> users = new ArrayList<>();
        HashMap<String, String> hm = new HashMap<>();
        
        for(int i=0; i<record.length; i++){
            String[] chat = record[i].split(" ");
            if(!chat[0].equals("Change")){
                users.add(new User(chat[0], chat[1]));
            }
            if(!chat[0].equals("Leave")){
                hm.put(chat[1], chat[2]);
            }
        }
        
        String[] answer = new String[users.size()];
        
        for(int i=0; i<users.size(); i++){
            User user = users.get(i);
            String str = hm.get(user.uuid);
            
            if(user.command.equals("Enter")){
                str += "님이 들어왔습니다.";
            }else{
                str += "님이 나갔습니다.";
            }
            answer[i] = str;
        }
        
        return answer;
    }
}
```
