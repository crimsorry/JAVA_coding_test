# 마법의 엘리베이터

## 내 풀이

```python
class Solution {
    public int solution(int storey) {
        int answer = 0;

        while (storey > 0) {
            int digit = storey % 10;
            int next = (storey / 10) % 10;

            if (digit < 5) {
                answer += digit;
            } else if (digit > 5) {
                answer += (10 - digit);
                storey += 10;
            } else { 
                if (next >= 5) {
                    answer += 5;
                    storey += 10;
                } else {
                    answer += 5;
                }
            }

            storey /= 10;
        }

        return answer;
    }
}
```
