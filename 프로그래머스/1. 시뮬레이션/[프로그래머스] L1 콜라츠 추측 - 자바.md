# 콜라츠 추측

https://school.programmers.co.kr/learn/courses/30/lessons/12943

## **내 풀이**

```java
class Solution {
    public int solution(int num) {
        int answer = 0;
        long n = num;
        
        while(n != 1){
            if(n % 2 == 0){
                n /= 2;
            }else{
                n = (n * 3) + 1;
            }
            answer++;
            if(answer >= 500 && n != 1){
                answer = -1;
                break;
            }
        }
        
        return answer;
    }
}
```
