## **내 풀이**

```java
import java.util.*;

class Solution {
    public int solution(int[] money) {
        int answer = 0;
        int[] dp1 = new int[money.length]; // 2부터 털기
        int[] dp2 = new int[money.length]; // 1부터 털기
        dp1[0] = 0;
        dp1[1] = money[1];
        dp2[0] = money[0];
        dp2[1] = (dp2[0] > money[1]) ? dp2[0] : money[1];
        
        for(int i=2; i<money.length; i++){
            dp1[i] = Math.max(dp1[i-1], dp1[i-2] + money[i]);
            if(i == money.length-1) continue;
            dp2[i] = Math.max(dp2[i-1], dp2[i-2] + money[i]);
        }
        
        return Math.max(dp1[money.length-1], dp2[money.length-2]);
    }
}
```
