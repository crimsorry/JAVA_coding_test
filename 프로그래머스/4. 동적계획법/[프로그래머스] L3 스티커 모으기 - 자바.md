# 스티커 모으기

## **내 풀이**

```java
class Solution {
    public int solution(int sticker[]) {
        int answer = 0;
        if (sticker.length == 1) return sticker[0];
        if (sticker.length == 2) return Math.max(sticker[0], sticker[1]);
        
        int[] dp1 = new int[sticker.length]; // 첫번째 부터   
        int[] dp2 = new int[sticker.length]; // 두번째 부터
        dp1[0] = sticker[0];
        dp2[0] = 0;
        
        if(sticker.length > 1){
            dp1[1] = Math.max(dp1[0], sticker[1]);
            dp2[1] = sticker[1];
        }
        
        for(int i=2; i<sticker.length; i++){
            dp2[i] = Math.max(dp2[i-1], dp2[i-2] + sticker[i]);
            if(i == sticker.length-1) continue;
            dp1[i] = Math.max(dp1[i-1], dp1[i-2] + sticker[i]);
        }
        
        return Math.max(dp1[sticker.length-2], dp2[sticker.length-1]);
    }
}
```
