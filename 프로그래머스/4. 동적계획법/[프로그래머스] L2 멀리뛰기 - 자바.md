# 멀리뛰기

## **내 풀이**

```java
class Solution {
    
    private final int MOD = 1_234_567;
    
    public long solution(int n) {
        long prev1 = 1;
        
        if(n == 1) return prev1;
        
        long prev2 = 2;
        
        for(int i=3; i<=n; i++){
            long prev3 = (prev1 + prev2) % MOD;
            prev1 = prev2;
            prev2 = prev3;
        }
        
        return prev2;
    }
}
```
