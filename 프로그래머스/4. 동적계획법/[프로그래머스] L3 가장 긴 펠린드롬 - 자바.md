# 가장 긴 팰린드롬

https://school.programmers.co.kr/learn/courses/30/lessons/12904

## **내 풀이**

```java
class Solution {
    public int solution(String s) {
        int n = s.length();
        boolean[][] dp = new boolean[n][n]; // n ~ n 까지는 페린드롬 일까?
        int answer = 1;

        // 문자열 길이 1 'a' 'b'...
        for (int i = 0; i < n; i++) {
            dp[i][i] = true;
        }

        // 문자열 길이 2 'aa' 'bb' 'ab' ...
        for (int i = 0; i < n - 1; i++) {
            if (s.charAt(i) == s.charAt(i + 1)) {
                dp[i][i + 1] = true;
                answer = 2;
            }
        }

        // 문자열 길이 3 이상 'aba' 'abbb' ...
        for (int len = 3; len <= n; len++) {
            for (int l = 0; l + len - 1 < n; l++) {
                int r = l + len - 1;
                if (s.charAt(l) == s.charAt(r) && dp[l + 1][r - 1]) {
                    dp[l][r] = true;
                    answer = Math.max(answer, len);
                }
            }
        }

        return answer;
    }
}

```
